# Goal

**Роут:** `(tabs)/goal`

## Назначение

Постановка и отслеживание целей по силовому показателю — по одной на каждое упражнение.

## Сценарий (S5)

Выбрать упражнение → задать цель (target 1RM + date) → смотреть список всех целей с прогрессом.

Альтернативный путь: Exercise Detail → кнопка "Поставить цель" → форма с предзаполненным упражнением.

## Содержимое экрана

### Нет ни одной активной цели

- Пустое состояние с CTA "Добавить цель"
- Кнопка открывает форму создания цели

### Список активных целей

Каждая карточка цели содержит:

- Название упражнения
- Target 1RM и target date
- Current 1RM (рассчитывается по формуле Epley из последних подходов)
- `remainingKg` = target 1RM − current 1RM
- Прогресс-бар
- Кнопка удалить / изменить цель

### Карточка достигнутой цели (current 1RM ≥ target 1RM)

- Сообщение "Достигнуто"
- CTA для постановки новой цели на это же упражнение

## Форма создания / редактирования цели

- Упражнение (поиск и выбор из списка; если открыта из Exercise Detail — предзаполнено)
- Target 1RM
- Target date

## Ограничения

- Максимум **одна активная цель на упражнение** (при создании новой старая помечается неактивной)
- Без программ тренировок и рекомендаций

## Модель данных

| Сущность | Поля |
|----------|------|
| `Goal` | `id`, `userId`, `exerciseId`, `targetOneRm`, `targetDate`, `isActive`, `createdAt` |
| `Exercise` | `id`, `name` |
| `SetEntry` | `weightKg`, `reps` — для расчёта current 1RM |
