# Friends

**Роут:** `(tabs)/friends`

## Назначение

Список друзей пользователя и добавление новых.

## Содержимое экрана

- Поле / кнопка "Add friend" — поиск по handle
- Список принятых друзей (`Friendship.status = accepted`)

## Переходы

| Действие | Куда |
|----------|------|
| Тап на друга | `friends/[id]` (Friend Detail) |

## Приватность

Друзья видят метрики по упражнениям (1RM и ratio). Масса тела как число **не отображается** друзьям — используется только для расчёта ratio.

## Модель данных

| Сущность | Поля |
|----------|------|
| `User` | `id`, `name`, `handle` |
| `Friendship` | `fromUserId`, `toUserId`, `status` (pending / accepted) |
