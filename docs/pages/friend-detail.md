# Friend Detail

**Роут:** `friends/[id]`

## Назначение

Сравнение прогресса по упражнению с конкретным другом.

## Сценарий (S3)

Открыть друга → выбрать упражнение → график "я vs друг" → absolute/ratio

## Содержимое экрана

- Имя / handle друга
- Выбор упражнения
- График сравнения — 2 линии: "я" и "друг"
- Переключатель режимов: **absolute** / **ratio**

## Режимы графика

### Absolute
Ось Y — значение 1RM в кг.

### Ratio
Ось Y — \(1RM / bodyWeight\).

- Для текущего пользователя — `bodyWeight` из его `BodyWeightEntry`
- Для друга — ratio рассчитывается на стороне друга; масса тела друга как число не передаётся

## Переходы

| Действие | Куда |
|----------|------|
| Тап на упражнение | `exercise/[id]` (Exercise Detail) |

## Модель данных

| Сущность | Поля |
|----------|------|
| `User` | `id`, `name`, `handle` |
| `SetEntry` | `workoutExerciseId`, `order`, `weightKg`, `reps` |
| `BodyWeightEntry` | `userId`, `date`, `weightKg` |
