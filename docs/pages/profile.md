# Profile / Settings

**Роут:** `(tabs)/profile`

## Назначение

Настройки пользователя: ник (handle), единицы измерения, масса тела.

## Содержимое экрана

- **Ник (handle)** — задаётся один раз, изменить нельзя. Если ник ещё не задан: поле и кнопка «Сохранить». Если задан — только отображение текущего ника. По нику друзья добавляют пользователя («Добавить друга»). В БД хранится в `users.username`.
- **Единицы измерения:** kg (пока фиксированы).
- **Масса тела:**
  - Форма добавления записи: дата + вес
  - Список/история записей по необходимости

## Модель данных

| Сущность | Поля |
|----------|------|
| `User` | `id`, `name`, `username` (ник/handle), `units` |
| `BodyWeightEntry` | `userId`, `date`, `weightKg` |
