# Plans

**Роут:** `(tabs)/plans`

## Назначение

Главный экран вкладки Plans. Список сохранённых планов тренировок и точка входа для старта пустой тренировки.

## Содержимое экрана

- FAB "Start empty workout"
- Список планов пользователя (`WorkoutPlan`)

## Переходы

| Действие | Куда |
|----------|------|
| FAB "Start empty workout" | `workout/active` (пустая тренировка) |
| Тап на план | `plans/[id]` (Plan Detail) |

## Модель данных

| Сущность | Поля |
|----------|------|
| `WorkoutPlan` | `id`, `userId`, `name`, `updatedAt` |
